Selection Sort

Selection Sort is an in-place comparison sorting algorithm that divides the input list into two parts: the sublist of items already sorted and the sublist of items remaining to be sorted. Initially, the sorted sublist is empty and the unsorted sublist is the entire input list.

Example:

First Pass:

  • 64 25 12 22 11 34 90
  • 11 25 12 22 64 34 90
  • 11 12 25 22 64 34 90
  • 11 12 22 25 64 34 90
  • 11 12 22 25 34 64 90
  • 11 12 22 25 34 64 90

Second Pass:

  • 11 12 22 25 34 64 90
  • 11 12 22 25 34 64 90
  • 11 12 22 25 34 64 90
  • 11 12 22 25 34 64 90
  • 11 12 22 25 34 64 90

The time complexity of selection sort is O(n^2) as there are two nested loops.

The space complexity of selection sort is O(1).

The main advantage of selection sort is the simplicity of the algorithm.

The main disadvantage of selection sort is its time complexity. It is not suitable for large datasets.

Java Code:

public class SelectionSort {
    public static void main(String[] args) {
        int[] arr = {64, 25, 12, 22, 11, 34, 90};
        selectionSort(arr);
        System.out.println("Sorted array");
        printArray(arr);
    }

    static void selectionSort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            int min_idx = i;
            for (int j = i + 1; j < n; j++)
                if (arr[j] < arr[min_idx])
                    min_idx = j;
            int temp = arr[min_idx];
            arr[min_idx] = arr[i];
            arr[i] = temp;
        }
    }

    static void printArray(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n; i++)
            System.out.print(arr[i] + " ");
        System.out.println();
    }
}
