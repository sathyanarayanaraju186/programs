Bucket Sort

Bucket Sort is a sorting algorithm that works by distributing the elements of an array into a number of buckets. Each bucket is then sorted individually, either using a different sorting algorithm, or by recursively applying the bucket sorting algorithm.

Example:

First Pass:

  • 38 27 43 3 9 82 10
  • 3 9 10 27 38 43 82

The time complexity of bucket sort is O(n^2) in the worst case and O(n+k) in the best case, where n is the number of elements and k is the number of buckets.

The space complexity of bucket sort is O(n+k).

The main advantage of bucket sort is that it is efficient for small datasets.

The main disadvantage of bucket sort is that it is not suitable for large datasets.

Java Code:

public class BucketSort {
    public static void main(String[] args) {
        int[] arr = {38, 27, 43, 3, 9, 82, 10};
        bucketSort(arr);
        System.out.println("Sorted array");
        printArray(arr);
    }

    static void bucketSort(int[] arr) {
        int n = arr.length;
        int[] bucket = new int[n];
        for (int i = 0; i < n; i++) {
            bucket[i] = 0;
        }

        for (int i = 0; i < n; i++) {
            bucket[arr[i]]++;
        }

        int index = 0;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < bucket[i]; j++) {
                arr[index++] = i;
            }
        }
    }

    static void printArray(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n; i++) {
            System.out.print(arr[i] + " ");
        }
        System.out.println();
    }
}
